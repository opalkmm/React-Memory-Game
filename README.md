# React-Memory-Game


*Titan Memory Game*

Click each Titan from "Attack on Titan" once and only once!

If you click the same titan twice you will lose.

Try to see if you can click them all!

Hosted Live on Heroku: https://titan-memory-game.herokuapp.com/
